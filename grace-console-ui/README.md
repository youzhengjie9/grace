# grace-console-ui


## 基本插件安装

### 安装vue2最新稳定版

```
npm install vue@^2
```

### 安装element-ui最新版

```
npm i element-ui -S
```

### 安装vuex3.0
```
npm i vuex@3
```

### 安装vue-router3.0
```
npm i vue-router@3
```

### 安装sass

```
npm install sass@1.26.8 --save-dev
```

```
npm install sass-loader@8.0.2 --save-dev
```


### 安装axios和vue-axios
```
npm install --save axios vue-axios
```


### 安装less
```
npm i less less-loader@7
```

### 安装vue-fragment
```
npm install vue-fragment --save
```

### 安装nprogress进度条的插件
```
npm i -S nprogress
```

### 安装echarts 
```
npm install echarts --save
```

### 安装moment时间格式化插件

```
npm install moment
```
